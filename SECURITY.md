## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly by opening a GitHub issue or by emailing adiratna89@gmail.com directly.

### What to Expect
- You will receive an acknowledgment within 48 hours
- We will investigate and respond with next steps
- If the vulnerability is accepted, we will work on a fix and notify you when it is resolved
- If the vulnerability is declined, we will provide an explanation

### Guidelines
- Please do not disclose the vulnerability publicly until we have had a chance to address it
- Provide detailed information to help us understand and reproduce the issue
- Be patient and allow us time to respond and fix the issue

Thank you for helping keep this project safe for everyone!
