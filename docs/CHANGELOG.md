# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-06

### Added
- Initial implementation of Naive Bayes Algorithm (GaussianNB) on the Iris dataset
- Complete Jupyter Notebook with:
  - Data exploration and visualization (pairplot, distribution plots)
  - Data preprocessing (train-test split, feature scaling)
  - Model training with Gaussian Naive Bayes classifier
  - Model evaluation (accuracy, confusion matrix, classification report)
  - Performance analysis and interpretation
- Repository documentation (README.md, CONTRIBUTING.md, CHANGELOG.md)
- Issue templates (Bug Report, Feature Request)
- Pull Request template
- `.gitignore` file for Python projects
- `requirements.txt` with project dependencies
- MIT License
- Folder structure: `notebooks/`, `data/`, `docs/`
